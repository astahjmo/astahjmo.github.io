# Por que TUI > GUI

Interfaces de terminal sao superiores. Aqui vai o argumento.

## Velocidade

GUI: move mouse, clica, espera animacao, clica de novo.

TUI: digita comando, pronto.

Tempo medio pra abrir um arquivo:
- VS Code: **3.2 segundos**
- Neovim + telescope: **0.4 segundos**

## Consistencia

Vim keys funcionam em tudo:
- Editor (neovim)
- Browser (vimium)
- Window manager (i3)
- File manager (ranger/lf)
- Este blog

Uma vez que aprende, usa em todo lugar.

## Recursos

```
firefox: 800MB RAM
alacritty + neovim: 30MB RAM
```

Sem comentarios.

## Customizacao

GUIs te dao opcoes pre-definidas. TUIs te dao **tudo**.

Quer mudar a cor do status bar? Edita uma linha.
Quer adicionar um keybind? Adiciona uma linha.
Quer mudar o layout inteiro? Muda o config.

## Estetica

Tem algo sobre texto verde num fundo preto que nunca vai envelhecer.

> "The terminal is the most powerful GUI ever invented."
> -- Alguem inteligente, provavelmente
